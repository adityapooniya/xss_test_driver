XSS Testing Framework

v3 Beta

This Website is a XSS test runner, each XSS redirects the browser to the next test if the XSS Vector works. All XSS vectors are safe (no malicious payloads etc... i swear) but your browser might crash sometimes, so save your tabs before using it, just in case If the XSS Fails (which is a good thing somehow) you might need to go to /test/next to step to the next Test.

  • Available Vectors
  • Registered browsers
  • Test your Browser(you must be registered first)
  • Resume Testing / Add running test tab
  • View your Results (slow)
  • View All test Results (slow)
  • Search for specific Results (can be awefully slow)
